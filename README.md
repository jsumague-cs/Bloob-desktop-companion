# Bloob-desktop-companion
People often procrastinate because tasks feel overwhelming. Existing productivity tools can be cold, complicated, or stressful.  Bloob aims to reduce this friction by combining intelligent reminders, conversational AI, and an engaging companion into one application.

Bloob came from his name "Blooberry" he is an alien that studied how humans live and ultimately wanted to help them to succeed in life by ensuring his human is productive.
